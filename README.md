# sebowebsite

SEBO 官方网站 V6 的 Vue 3 响应式迁移项目，目标是完整兼容手机和 PC，并保留现有中英文路由、视觉内容、SEO 与在线表单能力。

Vue 3 迁移代码位于 `02_完整工程源码`，开发分支为 `feat/vue3-migration`。当前交付包包括：

- `01_离线可直接浏览网站`：原始离线浏览版本
- `02_完整工程源码`：Vue 3 + TypeScript + Vite 完整工程源码
- `04_验收与路由清单`：路由、版本与原始验收记录
- `docs/superpowers/specs`：Vue 3 迁移设计文档

Vue 3 迁移设计见 [`docs/superpowers/specs/2026-07-15-vue3-migration-design.md`](docs/superpowers/specs/2026-07-15-vue3-migration-design.md)。
